Responda las siguientes preguntas

1. ¿Qué es Flexbox ?

a. Es un protocolo de comunicación utilizado en la conexión de dispositivos IoT (Internet de las cosas).
b. Es un motor de renderizado de páginas web utilizado por los navegadores modernos para mostrar contenido web.
c. Técnica de diseño web que permite crear diseños flexibles y adaptables en CSS.

2. ¿Qué significa "flex-direction" en Flexbox ?

a. Es una propiedad en Flexbox que define la dirección en la que se alinean los elementos flexibles.
b. Es una forma de dirigir la energía de tu página web hacia una dirección específica, lo que aumenta su velocidad y eficiencia.
c. "flex-direction" es una técnica avanzada de diseño web que te permite crear páginas con una estética elegante y fluida, similar a la de un acuario.

3. ¿Cómo se define un contenedor flexible en Flexbox ?

a. Se define como un recipiente que puede cambiar de forma y tamaño para adaptarse a cualquier contenido que se le agregue.
b. Se define un contenedor flexible estableciendo la propiedad "display" en "flex" en el elemento padre.
c. Un contenedor flexible en Flexbox se define envolviendo tu contenido en una caja mágica que se expande y contrae según tus necesidades.

4. ¿Qué propiedad se utiliza para alinear los elementos verticalmente ?

a. Align-items
b. Flex-direction
c. Flex-wrap

5. ¿Cómo se ajusta el tamaño de los elementos en Flexbox ?

a. Aplicando la propiedad "shrink-to-fit" que reduce el tamaño de los elementos a medida que el contenedor se hace más pequeño, manteniendo la proporción original.
b. Utilizando la propiedad "stretch-o-matic" que estira automáticamente los elementos de manera proporcional al espacio disponible en el contenedor.
c. Los elementos se ajustan utilizando la propiedad "flex-grow" y "flex-shrink".
